# Plugins

This directory stores repo-local Codex plugin source packages for KK Dev Skeleton productization.

Current plugin:

- `kk-dev-skeleton-adoption`: wraps the natural-language adoption workflow and V1 internal installer helper.

Marketplace rollout docs:

- `PARTNER_INSTALL.md`: copy-ready partner install prompt for the published Git marketplace.
- `MARKETPLACE_INSTALL.md`: admin install and upgrade flow.
- `MARKETPLACE_ROLLOUT.md`: seven-day rollout plan for pilot promotion.
- `ADMIN_INSTALL_CHECKLIST.md`: pre-install checks, install steps, and stop conditions.
- `PILOT_FEEDBACK.md`: pilot feedback form.

Published Git marketplace:

- `https://github.com/duanxibo/kk-ai-dev-skeleton.git`

This directory is not a marketplace by itself. Do not create or modify marketplace files unless a user explicitly asks for an install or publishing workflow.
